    • Atom常用插件整理

    2017-04-20技术探讨 开发工具 atom

    Atom 是 github 专门为程序员推出的一个跨平台文本编辑器。具有简洁直观的图形用户界面，并有很多有趣的特点：支持 CSS，HTML，Javascript等网页编程语言。它支持宏，自动完成分屏功能，集成文件管理器。

    必备插件

    linter

    linter 是一套 atom 的提示器，用以监视代码的质量，提示帮助信息。

    file-icons

    file-icons 是替换 Atom 的文件图标插件，让侧边栏的文件更加清晰。

    minimap

    minimap 在侧边展示迷你代码缩图，可以快速的定位到指定区域。安装相应插件后可以展示更多有用的信息。

    优秀插件

    emmet

    emmet 不用多说了，zen-coding写法，前端都懂

    linter-eslint

    linter-eslint 是用来提示 eslint 规则的信息

    linter-markdown

    linter-markdown 是用来提示 markdown 规则的信息

    pigments

    pigments 用来标记代码中的颜色

    minimap-pigments

    minimap-pigments 在 minimap 上标记代码中的颜色

    minimap-git-diff

    minimap-git-diff 在 minimap 上标记出代码的变更状态，新增的代码为绿色，修改的内容为黄色，删除的内容为红色，可以快速定位修改的地方。

    platformio-ide-terminal

    platformio-ide-terminal 避免了项目开多了终端控制台混乱，可以在 atom 中直接启用终端程序，方便又好用。

    docblockr

    docblockr 快速生成注释

    editorconfig

    editorconfig 在我看来是必备的，但是貌似使用的人比较少，所以列到优秀插件里了，主要是规范项目的编辑器格式，规范。当多人项目协作开发时，编辑器各不相同，为了保证代码风格，缩进一致，所以需要 editorconfig 来统一。

    autocomplete-module

    autocomplete-module require 模块时自动补齐模块名

    其他插件

    react

    react react 语法支持，react 开发必备插件

    language-vue

    language-vue vue 模板支持，开发 vue 项目必备插件
