    2017-05-18前端技术 css

    什么是 CSScomb ？

    CSSComb 是专门用来美化CSS代码的工具，可以定义自己的CSS风格，一键 format CSS代码。

    官方介绍：

    Makes your code beautiful

    支持多种IDE工具,Sublime/Atom/brackets

    详细配置说明看这里。

    如何使用

    在项目目录下建立.csscomb.json配置文件。添加指定配置内容。可以在csscomb的官网根据自己的个人习惯在线生成指定的配置内容。
